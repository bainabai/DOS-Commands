1. CD - Change directory
2. DIR - List directory contents
3. MD - Make directory
4. RD - Remove directory
5. COPY - Copy files
6. REN - Rename files
7. DEL - Delete files
8. TYPE - Display file contents
9. EDIT - Open text editor
10. FC - Compare files
11. FIND - Search for a text string in files
12. XCOPY - Extended copy
13. TREE - Display directory tree
14. CHKDSK - Check disk
15. FORMAT - Format disk
16. MOVE - Move files
17. ATTRIB - Display or change file attributes
18. FC - Compare two files or sets of files
19. MORE - Display output one screen at a time
20. PROMPT - Change the command prompt
21. VOL - Display a disk label
22. VER - Display the DOS version
23. SET - Display, set, or remove environment variables
24. TIME - Display or set the system time
25. DATE - Display or set the system date
26. CLS - Clear the screen
27. EXIT - Quit the CMD.EXE program
28. FOR - Run a specified command for each file in a set of files
29. IF - Execute a command conditionally
30. LABEL - Create, change, or delete the volume label of a disk
